ЗАВДАННЯ

У звітному HTML-документі створити html-розмітку, яка складається з форми логіна. Обробка відправлення форми form.login-form повинна відбуватися за подією submit. Під час відправлення форми сторінка не повинна перезавантажуватися. Якщо при сабміті у формі є незаповнені поля, виводь alert з попередженням про те, що "All form fields must be filled in". Якщо всі поля заповнені, збери значення полів в об'єкт, виведи в консоль і очисти форму.

PROGRAM

ЛОГІН

JS CODE

            document.querySelector('.login-form').addEventListener('submit', function(event) {
                event.preventDefault(); // Забороняємо перезавантаження сторінки

                const form = event.target;
                const email = form.elements.email.value.trim();
                const password = form.elements.password.value.trim();
                const messageBox = document.querySelector('.message');

                if (!email || !password) {
                    alert('All form fields must be filled in');
                    return;
                }

                const formData = {
                    email: email,
                    password: password
                };

                console.log(formData);

                form.reset(); // Очищаємо поля форми
            });

            <section class="form__section">
